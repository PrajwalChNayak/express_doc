Introduction to Express.js

What is Express.js?

Express.js is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. It's designed for building web applications and APIs quickly and easily. Express.js makes it easier to organize your application's functionality with middleware and routing.

Express.js has been called the de facto standard server framework for Node.js. It's designed to be simple, unopinionated, and flexible, allowing developers to structure their applications however they see fit.

Installation

To install Express.js, you need Node.js installed on your system. Once you have Node.js, you can create a new project and install Express using npm:

$ mkdir myapp
$ cd myapp
$ npm init -y
$ npm install express

Hello World Example

Here's a simple "Hello World" example to get started with Express.js:

const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

Save this code in a file named app.js in your project directory. Then run it with node app.js. Open your browser and navigate to http://localhost:3000 to see "Hello World!" displayed.

Key Features

  • Robust routing system for handling HTTP requests
  • Middleware support for processing requests
  • Template engine integration for generating HTML
  • Error handling capabilities
  • Static file serving
  • HTTP utility methods for building responses
  • Content negotiation
  • Performance optimizations

Express vs. Node.js HTTP Module

While Node.js provides the built-in http module for handling HTTP requests and responses, Express.js builds on top of it to provide a more intuitive interface, middleware capabilities, and a routing system, making web application development much easier.

Node.js HTTP Module

const http = require('http');

const server = http.createServer((req, res) => {
  if (req.url === '/' && req.method === 'GET') {
    res.writeHead(200, { 'Content-Type': 'text/plain' });
    res.end('Hello World!');
  }
});

server.listen(3000);

Express.js

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(3000);
