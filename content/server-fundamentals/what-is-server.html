Basic Definition: What is a Server?

A server is simply a computer that provides services to other computers. Imagine a library where you ask the librarian for a specific book - the librarian finds it and gives it to you. That's exactly what a server does! When you visit a website, send an email, or watch a video online, you're asking a server to provide that service for you.

ğŸ’¡ Key insight: Servers are designed to be "always on" and ready to help. While your personal computer might sleep or turn off, servers work 24/7 to make sure you can access websites, apps, and online services whenever you need them.

What action you want (GET = fetch, POST = send)

Success! (404 = not found, 500 = server error)

The actual data you requested

Simple Server Example

Here's a basic example that shows how a server works (don't worry about understanding every detail - just see the main idea):

server.js 
const http = require('http');
const url = require('url');

const server = http.createServer((req, res) => {
  const parsedUrl = url.parse(req.url, true);
  const path = parsedUrl.pathname;
  const method = req.method;

  // Set response headers
  res.setHeader('Content-Type', 'application/json');
  res.setHeader('Access-Control-Allow-Origin', '*');

  // Route handling
  if (path === '/' && method === 'GET') {
    res.statusCode = 200;
    res.end(JSON.stringify({ 
      message: 'Welcome to our API',
      timestamp: new Date().toISOString()
    }));
  } else if (path === '/users' && method === 'GET') {
    res.statusCode = 200;
    res.end(JSON.stringify({ 
      users: [
        { id: 1, name: 'John Doe' },
        { id: 2, name: 'Jane Smith' }
      ]
    }));
  } else {
    res.statusCode = 404;
    res.end(JSON.stringify({ 
      error: 'Route not found' 
    }));
  }
});

const PORT = process.env.PORT || 3000;
server.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

ğŸ’¡ Key Takeaway: This example shows a server listening for different requests (like asking for the homepage or user list) and responding with the right information - just like how a waiter takes different orders and brings the correct food!
