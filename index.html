Welcome to Express.js - A Beginner's Guide

What is a Server?

A server is like a computer program that waits for requests and sends back responses. Think of it as a restaurant waiter - it takes orders from customers (web browsers or apps), delivers those orders to the kitchen (processes the requests), and brings back food (data) to the customers.

What is an API?

API stands for Application Programming Interface. It's a set of rules that allows different software applications to communicate with each other. Imagine an API as a menu in a restaurant - it lists all the dishes (functions) available, how to order them (required parameters), and what you'll get in return (response data).

What is a REST API?

REST (Representational State Transfer) is a popular style for building APIs on the web. REST APIs use standard HTTP methods:

  • GET Fetch information (like viewing a webpage)
  • POST Create new data (like submitting a form)
  • PUT/PATCH Update existing data
  • DELETE Remove data

These familiar actions make REST APIs easy to understand and work with.
